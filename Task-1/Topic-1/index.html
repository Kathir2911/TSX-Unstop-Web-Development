Topic 1: The Core Trio - HTML Structure, Basic CSS Styling, and Introduction to JavaScript

HTML STRUCTURE

HTML (HyperText Markup Language) is the standard language used to create web pages. A basic HTML document starts with <!DOCTYPE html> to define the document type. It contains two main sections: the <head> and the <body>. The <head> includes meta-information like the page title and links to stylesheets, while the <body> contains the visible content such as headings, paragraphs, images, and links. All content is enclosed within the <html> tag, which marks the start and end of the document.

BASIC CSS STYLING

Basic CSS styling is used to change the appearance of HTML elements on a webpage. With CSS (Cascading Style Sheets), you can control the layout, colors, fonts, spacing, and more. For example, you can change text color using color, set background with background-color, align text with text-align, and apply margins or padding to elements. CSS can be written inline, internally within a <style> tag, or externally in a separate .css file linked to the HTML. This helps make web pages more attractive and easier to read.

INTRODUCTION TO JAVASCRIPT

JavaScript is a versatile, high-level programming language primarily used to create interactive and dynamic content on websites. It runs directly in web browsers, allowing developers to build features like animations, form validations, and real-time updates. Alongside HTML and CSS, JavaScript is one of the core technologies of web development. Its ease of use, vast ecosystem, and ability to work on both the front end and back end (with environments like Node.js) make it one of the most popular and widely used programming languages today.
